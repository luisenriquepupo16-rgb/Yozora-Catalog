# Yozora Store

## 📱 Descripción

Yozora Store es un catálogo de aplicaciones desarrolladas por Yozora 
Una tienda virtual donde puedes descargar todas mis aplicaciones en un solo lugar.

---

## ✨ Características

- ✅ Catálogo de aplicaciones por categorías
- ✅ Descarga directa de APKs
- ✅ Funciona sin internet (modo offline)
- ✅ Actualizaciones automáticas desde GitHub

---

## 🛠️ Tecnologías

- Kotlin + Jetpack Compose
- WebView con caché offline
- GitHub Pages como backend
